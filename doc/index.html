Get started

    API Endpoint

        http://localhost:8080

This API provides key value store for you. You can send data here with certain endpoints.

To use this API, you need to install the installation in your local file. Please go to the README.MD file for this.

PARAMETERS


# Here is a cURL example
curl --location --request GET 'http://localhost:8080/keys/all'

GET => To get the all keys you need to make a GET call to the following url :
http://localhost:8080/keys/all 



Result example :
{"balli":"3","emre":"2","kemal":"1"}
                

# Here is a cURL example
curl --location --request GET 'http://localhost:8080/keys?key=istanbul'

GET => To get a key you need to GET the following url with a parameter. :
http://localhost:8080/keys?key=istanbul 



Result example :
34
                

# Here is a cURL example
curl --location --request PUT 'http://localhost:8080/keys?key=istanbul&value=34'

PUT => To add a key value to the url you need to call the PUT method with parameters. :
http://localhost:8080/keys?key=istanbul&value=34 



Result example :
Key Added => istanbul
                

# Here is a cURL example
curl --location --request DELETE 'http://localhost:8080/keys'

DELETE => To delete all keys you need to DELETE the url. :
http://localhost:8080/keys 



Result example :
All datas was deleted.
                

# Here is a cURL example
curl --location --request DELETE 'http://localhost:8080/keys?key=testKey'

DELETE => To delete a key you need to DELETE the url. :
http://localhost:8080/keys?key=testKey 



Result example :
 Deleted key is: testKey

Errors

Key-Value Store API uses the following error codes:

Error Code Meaning
X000 No data in memory.
X001 Url Param 'key' is missing.
X002 There is no value for the key you called.
X003 Method not allowed.
